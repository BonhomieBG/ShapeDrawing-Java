# ShapeDrawing-Java
This Application use JavaFX as the main GUI handler. Provide a simple UI to draw 1D and 2D shapes. Other feature include: 
- Real Time Shape detail and Side Bar status update
- Undo and Redo Shapes
- Manual And Free Range drawing mode
- Quick Colors setting
- Linear Gradient Colors Support
- Display Coodinate at Location of Hovering
- Display Attribute of Shapes that is Hovering on
- Auto Draw shapes based on attribute provided
- Save and Load Feature (.Shape file extension)

Any Feedback regarding the project are welcome. CPP version of this Application available in my Github Repo.
